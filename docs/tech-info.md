# Tech Information

## Runtime

- Node.js (LTS recommended)
- npm 10+

## Web Stack

- React 19
- Vite 8
- TypeScript 5
- Tailwind CSS 3
- React Router 7
- Framer Motion
- Google Identity Services via `@react-oauth/google`
- Zustand
- Socket.IO client

## Backend Stack

- Express 5
- Socket.IO 4
- Mongoose
- JWT (`jsonwebtoken`)
- Cloudinary SDK
- Multer

## Data and Infra

- MongoDB
- Cloudinary (media)
- Optional Redis-ready patterns for scaling presence/events

## Build Commands

### Web

```powershell
cd web
npm run dev
npm run build
npm run preview
```

### Server

```powershell
cd server
npm run dev
npm run test
```
